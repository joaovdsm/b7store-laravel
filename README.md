# b7store-laravel
Projeto do módulo 18 do curso de Laravel da B7WEB (Clone OLX)

Recursos básicos (CRUD's):
- Criação de conta
	- Registro:
		- Nome
		- Email
		- Senha

	- Login:
		- Email
		- Senha

  - Perfil:
		- Nome
		- Email
		- Senha
		- Estado


- Criação de anuncios:
	- Slug
	- Titulo
	- Descrição
	- Preço
	- Estado
	- Categoria
		- Icone
		- Texto
	- Foto (Principal / Secundárias)
	- Negociável ou não (bool)
	- user_id
	- Entrar em contato
	- Visualizações
	- created_at
